
## 기술 스택
### 백엔드
* Spring Boot 3.3.0, Java 17
* JPA & QueryDsl
* Spring Security
* H2 (로컬, 테스트)
* Mysql (개발 서버)
* Embedded-Redis (로컬, 테스트)
* Redis (개발 서버)
* OAuth2 & JWT
* Actuator, Prometheus, Loki (모니터링)
* Spring Event


### 프론트
* Vue.js 3
* Vue Router
* Vuex
* Axios
* Chart.js
* jwt-decode


### 인프라
* AWS EC2
* Elastic Cache
* AWS S3 , S3 Bucket
* AWS RDS
* AWS Certificate Manager
* Route53
* Cloud Front
* Docker, Docker-compose
* Github Actions
* Nginx
* Let`s Encrypt
